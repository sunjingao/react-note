# react-note

> 使用react + redux 实现简单的记事本功能。

## technology stack

* [react](https://facebook.github.io/react/)                          
一个用于构建用户界面的 JAVASCRIPT 库。
* [redux](http://www.redux.org.cn/)   
react的状态管理容器。


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start
```
